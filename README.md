# Smart Route Management & Scheduling System (SRMSS)

## Project Overview
The Smart Route Management & Scheduling System (SRMSS) is an ASP.NET Core MVC web application developed to manage bus routes, schedules, drivers, buses, and users efficiently.

## Technologies Used
- ASP.NET Core MVC
- C#
- SQL Server
- Entity Framework Core
- Bootstrap

## Database
Use the `SRMSSDB.bak` or `SRMSSDB.sql` file to restore the database before running the project.
